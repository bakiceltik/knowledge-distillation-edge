\begin{abstract}
Accurate image-based plant disease classification can support precision
agriculture, but high-capacity convolutional networks are not always suitable
for edge or mobile deployment. This work evaluates whether knowledge
distillation can transfer the predictive behavior of a ResNet50 teacher to a
compact MobileNetV3-Small student on the 38-class PlantVillage dataset. We
train a supervised ResNet50 teacher, a supervised MobileNetV3-Small baseline,
and a distilled MobileNetV3-Small student using softened teacher predictions.
On the held-out test split, the distilled student reaches 0.9937 accuracy and
0.9907 macro-F1 with 1.56M parameters, compared with 23.59M parameters for the
ResNet50 teacher. These results indicate that, in this controlled dataset
setting, distillation can preserve near-teacher classification performance in a
model that is about 15.1 times smaller.
\keywords{Knowledge distillation \and Plant disease classification \and
MobileNetV3 \and ResNet50 \and Edge inference.}
\end{abstract}

\section{Introduction}

Plant disease recognition is a practical computer vision problem with direct
relevance to crop monitoring and precision agriculture. Deep convolutional
neural networks can achieve strong classification accuracy on curated leaf
image datasets, but models with high parameter counts may be inconvenient for
mobile or edge-oriented deployment. This creates a tension between predictive
accuracy and computational efficiency.

This project studies that trade-off through knowledge distillation. A
high-capacity ResNet50 teacher is trained on the full PlantVillage dataset and
then used to guide a lightweight MobileNetV3-Small student. The aim is not to
claim deployment on a specific device, but to evaluate whether a compact
student can retain the teacher's behavior while reducing parameter count and
maintaining low batch-one inference latency.

The contributions are as follows. First, we train a ResNet50 teacher on the
full 38-class PlantVillage task. Second, we train a supervised
MobileNetV3-Small baseline under the same split and preprocessing pipeline.
Third, we distill the ResNet50 teacher into MobileNetV3-Small using hard
labels and softened teacher logits. Finally, we compare accuracy, macro-F1,
parameter count, and measured latency across the teacher, supervised student,
and distilled student.

\section{Background and Related Work}

PlantVillage provides a widely used collection of labeled plant leaf images
for plant health research \cite{hughes2015open}. The dataset has enabled
controlled studies of image-based plant disease classification, although its
curated imaging conditions are cleaner than many field scenarios.

Knowledge distillation was introduced as a way to transfer information from a
larger teacher model to a smaller student model \cite{hinton2015distilling}.
Instead of learning only from one-hot labels, the student also learns from the
teacher's softened class probabilities, which can encode similarities between
classes. This makes distillation attractive when the deployment model must be
smaller than the model used to generate the strongest predictions.

The teacher in this work is ResNet50, a residual network architecture that
uses skip connections to ease optimization of deeper convolutional models
\cite{he2016deep}. The student is MobileNetV3-Small, an architecture designed
for mobile inference through efficient convolutional blocks and
hardware-aware design choices \cite{howard2019searching}. The combination is
therefore natural for an edge-oriented compression experiment: ResNet50
provides a strong teacher, while MobileNetV3-Small provides a compact student
model.

\section{Methodology}

Let $z_t$ and $z_s$ denote the teacher and student logits for an input image,
and let $y$ denote the ground-truth class label. The supervised teacher and
supervised student baselines are trained with cross-entropy against the hard
labels. For distillation, the teacher is first trained and then frozen. The
student is optimized using a weighted sum of the cross-entropy loss and a
temperature-scaled Kullback--Leibler divergence term:

\begin{equation}
  \mathcal{L} =
  \alpha T^2
  \mathrm{KL}\left(
    \mathrm{softmax}\left(\frac{z_t}{T}\right)
    \,\middle\|\,
    \mathrm{softmax}\left(\frac{z_s}{T}\right)
  \right)
  + (1-\alpha)\mathrm{CE}(y, z_s).
  \label{eq:kd-loss}
\end{equation}

In the final experiment, the distillation temperature is $T=4.0$ and the
distillation weight is $\alpha=0.5$. The $T^2$ factor follows the common
practice of compensating for gradient scaling introduced by softened
probabilities. This notation matches the implementation used in the active
distillation training script, where $\alpha$ controls the weight of the
teacher-guided term.

MobileNetV3-Small is used as the student because it has substantially fewer
parameters than ResNet50 while retaining a convolutional design appropriate
for image classification. This makes it a suitable candidate for studying
whether teacher supervision can preserve classification performance under a
smaller parameter budget.

\section{Experimental Setup}

Experiments use the full PlantVillage directory available in the repository.
The pipeline creates deterministic train, validation, and test splits with
seed 42. Images are resized to $224 \times 224$ pixels and normalized with
ImageNet statistics. Training uses random horizontal flipping and small random
rotations; validation and test evaluation use deterministic resizing and
normalization.

\section{Results and Discussion}

Table~\ref{tab:results} summarizes the final test-set results. The ResNet50
teacher reaches 0.9936 test accuracy and 0.9912 macro-F1. The supervised
MobileNetV3-Small baseline is already strong, with 0.9851 accuracy and 0.9779
macro-F1. After distillation from ResNet50, MobileNetV3-Small improves to
0.9937 accuracy and 0.9907 macro-F1 while retaining the same 1.56M parameter
count.

  \centering
  \caption{Final test results on the 38-class PlantVillage task.}
  \label{tab:results}
  \resizebox{\textwidth}{!}{

    MobileNetV3-Small & Supervised baseline & 0.9851 & 0.9779 & 1,556,806 & 3.51 ms \\

\end{table}

The distilled student uses approximately 15.1 times fewer parameters than the
teacher while matching it closely on accuracy and macro-F1. The latency of the
distilled student remains close to the supervised MobileNetV3-Small baseline
and below the measured ResNet50 latency in this local setup. These results
suggest that, for this dataset and split, distillation is effective not merely
as a compression technique but also as a way to recover much of the teacher's
predictive behavior in the smaller architecture.

The comparison between the supervised and distilled MobileNetV3-Small models
is particularly important because the architecture and parameter count are the
same in both rows. The improvement from 0.9851 to 0.9937 accuracy therefore
does not come from increasing student capacity. Instead, it indicates that the
teacher logits provide useful training information beyond the hard labels
alone. The macro-F1 improvement from 0.9779 to 0.9907 suggests that the gain is
not limited to majority classes, although per-class behavior should still be
inspected before drawing conclusions about deployment robustness.

The remaining errors are concentrated in visually related categories, such as
maize leaf-spot/blight classes and several tomato disease classes, where
symptoms can be similar at image level. This pattern supports using macro-F1
alongside accuracy, since class-level behavior matters in a multi-class plant
health setting.

The comparison should still be interpreted with caution. PlantVillage images
are relatively clean and controlled, which can make the task easier than
in-field plant disease recognition. The high supervised student result
supports this point: MobileNetV3-Small already performs well before
distillation. Therefore, the most defensible conclusion is that distillation
preserves very high accuracy in a much smaller model on PlantVillage, not that
the model is ready for unconstrained agricultural deployment.

Parameter counts clarify the deployment motivation. In the final 38-class
configuration, ResNet50 has 23,585,894 trainable parameters, whereas the
compact student has 1,556,806. The distilled student therefore keeps the
smaller model size of the supervised student while approaching the teacher's
predictive performance. In this setting, it is not meaningfully slower than
the supervised student, with measured latency of 3.53 ms compared with
3.51 ms. This small difference is expected because the teacher is used only
during training; inference uses the student alone.

\section{Reproducibility and Limitations}

The final experiments are configuration-driven. The teacher run corresponds to
the ResNet50 full-dataset configuration, the supervised student run
corresponds to the MobileNetV3-Small full-dataset configuration, and the
distillation run uses the ResNet50 teacher checkpoint saved under the full
teacher output directory. The reported test metrics are taken from the
corresponding result files in the repository outputs rather than copied from
terminal logs. This keeps the report tied to reproducible artifacts: each run
stores a model summary, training history, final metrics, a single-row result
CSV, and a confusion matrix.

Several limitations remain. First, the split is deterministic but generated
from the available PlantVillage directory rather than from an official
external benchmark split. Second, PlantVillage leaf images are comparatively
controlled; performance on field images with cluttered backgrounds, lighting
variation, occlusion, and different cameras may be lower. Third, the latency
measurements are local Apple Silicon MPS measurements. They are useful for
comparing the models in this environment, but they should not be interpreted
as measurements from a deployed mobile application or embedded accelerator.

The distillation hyperparameters were also kept fixed in the final comparison.
The selected temperature $T=4.0$ and weight $\alpha=0.5$ are reasonable
defaults and worked well in this setting, but a more complete study would vary
temperature, loss weighting, learning rate, and training length. Such tuning
would help separate the effect of the teacher architecture from the effect of
the distillation schedule.

\section{Conclusion}

This work evaluated knowledge distillation for edge-efficient plant disease
classification on the full 38-class PlantVillage dataset. A ResNet50 teacher
was distilled into a MobileNetV3-Small student, producing a compact model with
0.9937 test accuracy and 0.9907 macro-F1. The student has 1.56M parameters,
about 15.1 times fewer than the 23.59M-parameter teacher, and maintains low
local batch-one latency.

The results indicate that knowledge distillation can preserve near-teacher
performance in a substantially smaller model in this controlled setting. The
main limitation is dataset realism: PlantVillage is cleaner than many field
images, so future work should evaluate on more realistic plant disease
datasets, test on actual edge hardware, and tune the distillation temperature
and loss weighting more systematically.
